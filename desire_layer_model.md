# 有機体の維持から生まれる「欲望レイヤー」モデル
---

## 0. 目的

この文書は、「欲望」を **維持（環境変化を織り込んだ動的維持）** を実現するための **制御信号（ドライブ／方針）** として階層化したモデルを、議論参加者以外にも共有できる形に整理したものです。

- **低位（低級）**：短期・直接・反射に近い／媒介が少ない／脳の内側
- **高位（高級）**：長期・間接・モデル更新や規範／媒介が多い／脳の外側
- 階層は一方向の「上昇」だけでなく、**上位が下位を制御するフィードバック** も持つ（例：禁欲、断食、使命感）。

> 重要：階層は「順位」ではなく **条件（発火しやすさ・依存関係）**。
> 低位が不安定だと高位は発火しにくい一方、危機では高位が低位を支えることもあります。

### 立場表明（このモデルの前提）

このモデルは「社会（的活動）をこういう前提で見る」という **作業仮説**であり、数学の定理のような証明可能性を主張しない。ただし、恣意的な断定に閉じないために、次の立場を明示する。

- **着想源**：生体が外乱込みで安定を保つという **動的維持**（ホメオスタシス／アロスタシス）の考え方。
- **基本仮説**：欲望は、その動的維持を成立させるための **制御信号（理想状態からの誤差信号に対する方針）** として階層化できる。
- **連続化の狙い**：個体の維持（呼吸・安全など）と、共同体の維持（規範・制度・文化など）を、スケール違いの「制御」として連続的に記述する。
- **観測上の立場**：社会組織（企業・国家・共同体など）を、生物そのものではないが「境界・代謝・記憶・免疫」を持つ **擬似的な生命（自己維持する制御系）** として扱う。これは責任主体を免罪するためではなく、現象を制御構造として読み解くための比喩である。
- **構造的根拠**：レイヤーの順序は、脳の **内側から外側への構造**（脳幹→辺縁系→皮質）と対応させている。これは「科学」ではなく「科学っぽさ」の担保であり、OSI参照モデルのような階層的依存関係を説明するための補助線である。

### 評価規約（"それなりの正しさ"を保つために）

本モデルは、次の規約に基づいて改善される。

1. **反例が出たら更新する**（教義化しない）。
2. **説明／予測／設計に役立つか** で評価する（便利さを検証可能な価値として扱う）。
3. 同じ対象を扱う他モデルと比べ、**説明が短く・混線が少なく・共有しやすい** かを重視する。

## 1. 方針：欲望は「起源（誤差信号）」で階層（レイヤー）を決める

同じ行動でも、**何が鳴っているか（起源＝誤差信号）** で所属レイヤーが変わり得ます。
このモデルでは、**欲望レイヤー（L）を「行動」ではなく「起源」で決める** ことを明示します。

- **起源（レイヤー判定の対象）**：いま何を維持しようとしているか／どんな誤差が鳴っているか
- **戦略（行動）**：その誤差を埋めるために選ばれた具体行動（道具・制度・計画も含む）
- **自覚**：本人がその誤差を意識できるか（起源と自覚は一致しないことがある）

## 2. 補助タグ

### 2.1 スコープ：I / We（Weなら広さ）

- **I**：個体の維持
- **We-近**：場・対面の関係（家族、恋人、チーム、界隈）
- **We-中**：共同体・組織（会社、学校、地域など）
- **We-広**：制度圏・文化圏（国家、宗教圏、文明、世代）
- **I→We(橋)**：個体内の状態を共有可能な形へ変換して橋渡し

### 2.2 モード：Now / Better

- **Now**：今の破綻回避・可動域復帰
- **Better**：将来適応・不確実性低減・長期維持
- **Now→Better**：短期対応が長期改善にもつながる

### 2.3 頻度（タイムスケール）

各レイヤーに、**起源信号が強く立ち上がる代表スケール**を添えます（目安）。

### 2.4 強度と閾値

同じレイヤーの欲望でも、**誤差信号の強さには個人差がある**。

- **強度**：同じ刺激に対して、どのくらい強く鳴るか
- **閾値**：どのくらいの刺激で鳴り始めるか

強度・閾値の違いは、神経系の特性による個人差であり、それ自体は善悪ではない。「平均」との差が弊害になる（例：ADHD的特性＝一部の欲望が弱く出る、HSP的特性＝一部の欲望が強く出る）。

## 3. Tier構造：脳の内側から外側へ

レイヤーは5つのTier（層）にグループ化される。これは脳の内側から外側への構造と対応する。

| Tier | 名称 | 解剖学的部位（被りなしの割当） | レイヤー | 時間軸 |
|------|------|----------------------------------|----------|--------|
| **T0** | 基盤層 | （脳外：身体・環境・物理化学） | L0 | — |
| **T1** | 反射層 | 脊髄／延髄／橋／中脳（脳幹） | L1, L2, L3 | 秒〜日 |
| **T2** | 生存層 | 視床下部／扁桃体／中脳水道周囲灰白質（PAG） | L4, L5, L6 | 瞬時〜週 |
| **T3** | 関係層 | 上側頭溝（STS）／側頭頭頂接合部（TPJ）／前帯状皮質（ACC） | L7, L8, L9 | 分〜年 |
| **T4** | 予測層 | 背外側前頭前野（dlPFC）／眼窩前頭皮質（OFC）／海馬 | L10, L11, L12 | 週〜年 |
| **T5** | 物語層 | 内側前頭前野（mPFC）／後帯状皮質（PCC）／楔前部（precuneus） | L13, L14 | 月〜世代 |

### Tier間の依存関係

```
T1が壊れると → T2以上が機能しない（呼吸が止まれば死ぬ）
T2が不安定だと → T3以上が発火しにくい（飢餓・恐怖下で所属を考えられない）
T3が不安定だと → T4以上が発火しにくい（孤立していると蓄積・探索に向かない）
T4が不安定だと → T5が発火しにくい（備えがないと規範を考える余裕がない）
```

**ただし、逆方向のフィードバックもある。**

```
T5（規範・使命）がT2（生存）を上書きする例 → 殉教、ハンガーストライキ
T3（所属）がT4（蓄積）を駆動する例 → 仲間のために学ぶ
```

## 4. 欲望レイヤー（起源ベース）：L0〜L14

以下では、各レイヤーを「起源（誤差）」で定義し、具体行動は「戦略例」として添えます。

### T0:基盤層

#### L0：前提層（欲望なし）— 物理法則・化学反応（散逸／平衡化）

- **タグ**：`（非生命）` / `—` / `頻度: —`
- **対応構造**：骨格、結合組織、細胞外マトリクス
- **起源（誤差）**：なし（目標域がない）
- **要点**：放置すると、摩擦・拡散・反応により勾配がならされ、構造がほどける（散逸）。
- **直感例**：温度差が消える／混ざる／錆びる／乾く／崩れる。

### T1：反射層

#### L1：呼吸・飲水欲（最短ループ入力）

- **タグ**：`I` / `Now` / `頻度: 秒〜分（呼吸）、時間（飲水）`
- **対応構造**：延髄（呼吸中枢）、視床下部（渇中枢）
- **起源（誤差）**：酸素・水・電解質の欠乏（神経発火の三要素）
- **自覚される形**：息苦しさ、渇き
- **戦略例（行動）**：呼吸を整える／水を飲む／換気／場所移動／塩分補給
- **補足**：電解質の乱れは直接自覚しにくく、L8（調律）の不調として現れることが多い（だるさ、脱力、「なんか変」）

#### L2：排泄欲（出力・浄化）

- **タグ**：`I` / `Now` / `頻度: 時間`
- **対応構造**：脊髄反射、仙髄、橋（排尿中枢）
- **起源（誤差）**：内部の過剰・不要物・内圧が閾値を超える
- **自覚される形**：尿意、便意、吐き気
- **戦略例（行動）**：排尿・排便／嘔吐／発汗／衛生行動

#### L3：回復・保守欲（睡眠・休息）

- **タグ**：`I` / `Now→Better` / `頻度: 日（＋分〜時間の小休止）`
- **対応構造**：視床下部（視交叉上核）、松果体、脳幹網様体
- **起源（誤差）**：修復・統合・再起動の不足（疲労・回復遅延）
- **自覚される形**：眠気、疲労感
- **戦略例（行動）**：睡眠／休息／休暇／病欠／回復ルーティン
- **補足**：「休息」は"何の誤差を埋めるか"で起源が変わる：修復目的→L3、調律目的→L8

### T2:生存層

#### L4：食欲（入力・補給）
- **タグ**：`I` / `Now→Better` / `頻度: 時間（飲水）〜半日/日（摂食）`
- **起源（誤差）**：エネルギー／水分／塩分などの欠乏が進む
- **戦略例（行動）**：食べる／飲む／採餌・狩り／「すぐ食べられるもの」を確保する。

#### L5：痛み回避・安全欲（損傷回避）
- **タグ**：`I（主）` / `Now` / `頻度: 最短は瞬時（割り込み）`
- **起源（誤差）**：急性リスク（損傷・捕食・暴力・事故・寒暑など）が迫る
- **戦略例（行動）**：逃亡／回避／隠れる／防護（衣服・装備・施錠）／シェルター確保／服従（害回避）。

#### L6：性（発情）欲（生理ドライブ相）
- **タグ**：`I` / `Now（割り込み）` / `頻度: 日〜週（周期・状況依存）`
- **起源（誤差）**：発情・性的緊張として立ち上がる生理ドライブ
- **戦略例（行動）**：求愛・接近衝動／性的刺激の探索／緊張の解放（生理側の例）。
- **注記**：同意・関係・規範などの「行動選択」は起源が別（L8/L9/L13など）。

### T3:関係層

#### L7：情動・覚醒の調律欲（実行状態の維持）
- **タグ**：`I` / `Now` / `頻度: 分〜時間`
- **起源（誤差）**：過覚醒／低覚醒／情動の乱れで「動ける状態」から外れる
- **戦略例（行動）**：上げる（運動・外出・会話）／下げる（深呼吸・瞑想・入浴）／状態切替（散歩・ルーティン）。
- **補足（習慣）**：
  - 習慣は多くの場合、**L7（調律）を自動化した制御ループ**として立ち上がる。  
    例：朝のルーティン、同じ席に座る、音楽をかける、SNS確認、コーヒー、寝る前の決まった手順など。  
  - 目的は「迷い・不安・負荷」を減らし、可動域に入り続ける／戻ること。

#### L8：所属・摩擦低減欲（場の共同体の短期維持）
- **タグ**：`We-近` / `Now` / `頻度: 分〜日`
- **起源（誤差）**：排除・孤立・摩擦の増大（場が壊れる／協力が得られない）
- **戦略例（行動）**：空気を読む／共通話題／謝罪・フォロー／役割を演じる／権力に合わせる（場目的）。

#### L9：親密・ケア・養育欲（関係と世代の維持）
- **タグ**：`We-近→（世代）` / `Better` / `頻度: 日〜年`
- **起源（誤差）**：信頼・コミット・保護・育成の不足（関係が弱る／世代維持が崩れる）
- **戦略例（行動）**：看病・支援／傾聴／役割分担／子育て・教育／関係修復。
- **補足**：子育てには危険回避（起源L5）や覚醒反応（起源L7）が混ざり得るが、「世代維持」という持続的起源はL9に置く。

### T4:予測層

#### L10：技能・資源の蓄積欲（予測的維持）
- **タグ**：`I→We-中` / `Better` / `頻度: 週〜年`
- **起源（誤差）**：将来の損失・脆弱性・適応不足への予期的な誤差（“備えが足りない”）
- **戦略例（行動）**：学習・訓練／備蓄・保存食作成／貯蓄・保険・バックアップ／計画・仕組み化。
- **補足（技術面の習慣）**：技術面・手技面の「習慣化」は、ここでは **技能の蓄積（L10）**として扱う。  
  例：反復練習、作業手順の型化、チェックリスト運用、身体に染み込んだ所作。

#### L11：探索・好奇・モデル更新欲（維持のための変化）
- **タグ**：`I→We` / `Better` / `頻度: 日〜年`
- **起源（誤差）**：不確実性・未知・変化への脆弱性（“地図が足りない”）
- **戦略例（行動）**：旅／調査／研究／試作・実験／遊び（安全な探索）／創作。

#### L12：外化・表現欲（共有可能な信号化）
- **タグ**：`I→We(橋)` / `Now→Better` / `頻度: 必要時（分〜週）`
- **起源（誤差）**：内的状態・違和感・未整理情報が個体内に閉じて処理不能になる
- **戦略例（行動）**：日記・メモ・ログ／相談／説明資料化／作品化／儀礼化。

### T5:物語層

#### L13：規範・物語・制度欲（共同体の長期維持）
- **タグ**：`We-中→We-広` / `Better` / `頻度: 月〜年〜世代`
- **起源（誤差）**：集団の方針が揃わず、行動の一貫性が崩れる（長期の協調が壊れる）
- **戦略例（行動）**：ルール化／合意形成／教育・継承／理念・物語化／制度運用（裁定、罰と赦し）。
- **補足（不文律の浸透）**：制度面の「習慣化」は、ここでは **不文律（慣習）の浸透**として扱う。  
  例：業界慣行、暗黙の了解、礼節、空気、やってはいけないことの内面化。

#### L14：創造的破壊欲（共同体の動的維持）
- **タグ**：`We-広（ことが多い）` / `Better` / `頻度: 年〜十年〜（時に世代）`
- **起源（誤差）**：現行枠組みが環境変化に追従できず、長期維持が破綻する兆候が出る（“枠が限界”）
- **戦略例（行動）**：改革・刷新／基盤の置換／パラダイム転換／大規模再設計。

## 5. 「壁」：レベルが成立する前提（レベルの付記）
ここでいう「壁」は次を意味します。

> **壁＝その欲（起源信号）を自覚／満たして初めて、次のレベルの維持が“成立した”と言える境目**  
> （新しい維持対象が導入され、新しい誤差信号が生まれ、その信号を扱えるようになる）

以下では「どのレベルの前に置く壁か」を明記します。

### W0：非生命 ↔ 生命（L0とL1の間）
- **何が成立するか**：境界（膜）＋代謝＋負帰還制御によって「自己維持」が立ち上がる  
- **これが満たされて初めて**：L1以降の「欲望（制御信号）」が意味を持つ

### W1：全身統合の壁（単細胞 ↔ 多細胞）（L4とL5の間）
- **何が成立するか**：局所反応から、全身スケールの統合維持へ（役割分担・内部分業・全体最適）  
- **これが満たされて初めて**：外界に対する「身体まるごと」の安全・回避（L5）が厚みを持つ

### W2：系統（遺伝子）スケールの壁（個体維持 ↔ 系統維持）（L5とL6の間）
- **何が成立するか**：個体寿命を越えた最適化（繁殖・養育が“機能”として重くなる）  
- **これが満たされて初めて**：発情（L6）と養育（L9）が持続的な維持対象として立つ

### W3：社会性の登場（高等哺乳類：愛着・遊び・協力の壁）（L7とL8の間）
- **何が成立するか**：個体間の長期関係が維持対象になる（愛着・共感・協力）  
- **これが満たされて初めて**：
  - 所属・摩擦低減（L8）の誤差（排除・孤立・場の崩壊）が強く立つ  
  - 親密・ケア・養育（L9）が「関係維持」として厚くなる  
  - 遊びが探索（L11）を安全に回す入口になる

### W4：戦略性の登場（学習・予測・計画の壁）（L9とL10の間）
- **何が成立するか**：反射中心→学習→予測→計画（モデルを介した制御）  
- **これが満たされて初めて**：
  - 予測的維持（L10）＝「備えが足りない」という起源誤差が立ち上がる  
  - 探索更新（L11）＝「地図が足りない」という起源誤差が独立に意味を持つ

### W5：文化（宗教）の誕生（象徴・儀礼・物語の壁）（L11とL12の間）
- **何が成立するか**：象徴の共有／共同注意／儀礼／物語化（宗教を含む）が、集団の維持装置として働き始める  
- **これが満たされて初めて**：
  - 外化・表現（L12）が「共同体の操作可能な信号」になる  
  - 規範・物語（L13）が「協調の足場」になる（小規模でも可）

### W6：文明の誕生（制度・分業・外部記憶の壁）（L12とL13の間）
- **何が成立するか**：分業・交換・インフラ・外部記憶（記録、貨幣、法）が“維持装置”として立つ  
- **これが満たされて初めて**：規範・制度（L13）が個体を越えた長期維持の主戦場になる

### W7：自己決定社会（民主主義）の壁（加速・制度化・自己参照）（L13とL14の間）
- **何が成立するか**：印刷・教育・市場・メディア・科学技術により、象徴と制度が高速に複製・拡散・更新される  
- **これが満たされて初めて**：
  - L12（表現）が「産業・制度」と結びつき、自己参照的に進化しやすくなる（現代美術など）  
  - L14（創造的破壊）が“更新の通常運転”として頻発し得る

（注記）ここでの「自己決定社会（民主主義）」は時代区分ではなく、その国家/コミュニティに生きる人々全員が自己決定権（主権）および他者に異議申し立てする権利が法的もしくは慣習的に保証される性質を指す。加えて、自己決定社会はその自己決定権を本人／保護者の判断で委任・制限・一時停止（放棄）する自由も含む（ただし、退出・撤回・異議の回路が実質的に確保されることを条件とする）。現代にも非民主的な文明は併存し、将来には自己決定社会が主座でない秩序も成立し得る。

## 6. 迷いやすい境界：起源で見る最小ルール（任意）
- **同じ行動でも、何の誤差を埋めたいか？**で起源が決まる  
- 例：
  - 権力への追従：害回避（L5）／場の摩擦（L8）／規範内面化（L13）  
  - 貯蓄：欠乏（L4）／喪失（L5）／不安調律（L7）／予測的維持（L10）  
  - 休息：修復（L3）／調律（L7）  
  - 習慣：調律の自動化（L7）／技能の型化（L10）／不文律の浸透（L13）

## 7. 免責と読み方
- レイヤーは「固定の箱」ではなく、状況により複数起源が同時に鳴る（混合する）。  
- 重要なのは、**いま何を維持しようとしているか／誤差はどこか**を読み取れること。


## 8. 欲望レイヤーから見る、産業としての自己決定社会の美術（近現代美術）の役割
自己決定社会の美術（近現代美術）は「余暇」「娯楽」「趣味」として消費されることが多い。  
しかし本モデルでは、自己決定社会の美術を **社会（擬似生命）が動的維持（ホメオスタシス）を行うための産業**として位置づけ直す。

（用語注）ここでの「自己決定社会」は国家に限らずコミュニティにも適用される。便宜上「自己決定社会の美術（近現代美術）」と呼ぶが、非自己決定的な環境における外化・枠更新の実践も存在し得る。

### 8.1 結論（最短）
- 自己決定社会の美術は、主に **L12（外化・表現）**と **L14（創造的破壊）**に関わる誤差を扱う産業である。  
- 入口として **L7（情動・覚醒の調律）**を満たすことが多いが、社会的価値はそこだけに留まらない。

### 8.2 入口：L7（調律）としての「美しい」「気持ちいい」
- 「きれいなものを見たい」「気分を変えたい」はL7の誤差として自然であり、自己決定社会の美術もまずここで受容されうる。  
- これは「暇つぶし」ではなく、可動域に戻るための **調律インフラ**（軽量な回復）として機能する。

### 8.3 核：L12（外化）— 言語化できない誤差の“共有可能化”
自己決定社会の美術は、社会でうまく語れない違和感（個体内に閉じた誤差）を外へ出し、共有可能な信号に変換する。

- 個人の「説明できない」が、作品という形で共有できる「観測対象」になる  
- 共有されることで、議論・教育・批評・継承（L13）へ接続できる  
- 言語だけでなく、形・空間・身体・音・制度の操作によって外化が行われる

### 8.4 コア機能：L14（創造的破壊）— L13への反例提示と枠の更新
L13（規範・制度）は長期維持に必要だが、環境変化や制度疲労で必ず古びる。  
自己決定社会の美術は、**反例・皮肉・逸脱・アンチテーゼ**としてそれを可視化し、枠を更新する回路になる。

- 「この前提は本当に妥当か？」という問いを、作品として提示する  
- 禁忌・反逆・風刺・批評が、社会の“免疫反応”として作用する  
- 結果として、より **適応的な均衡（動的安定）**が回復されうる

### 8.5 L13との接続：制度・市場としての美術（産業化の意味）
自己決定社会の美術が産業である以上、L13（制度）と切り離せない。

- 美術館・ギャラリー・批評・教育・コレクターは、L12/L14の信号を「通る形」にする制度装置  
- 価格や評価は、しばしばL7の快さよりも **L13（権威・物語）×L14（更新度）**で決まりやすい  
- 「なぜ高いのか」「なぜ無視されるのか」は、買い手（You）の欲望と制度の接続の差として説明できる

### 8.6 産業としての意義（社会の低位維持と両立する）
呼吸・安全・補給（L1〜L5）を支える産業が社会の静的維持を担うなら、  
自己決定社会の美術は、社会の **動的維持**（L12/L14）を担う産業である。

- 「余暇」ではなく、長期的には社会の更新コストを下げ、崩壊を避けるための装置になりうる  
- したがって、美術の価値は「好き嫌い」だけでなく、社会が持つ更新能力としても評価できる

#### 使い方のメモ（用途への写像）
- 美術制作: L7→L12のブリッジ設計、L13整合、L14の更新度評価に各項目を参照。
- プロダクト/プロジェクトマネジメント: 予測処理・強化学習・必要多様性・レジリエンスをKPI/設計制約に反映。

- MVV制定: 文化進化・シグナリング・社会的アイデンティティで「通る言葉」と儀礼を設計。
- 社会制度/政党支持: オストロム・ノース・デュルケムでガバナンス/規範/連帯を評価・設計。
- 広報戦略: 予測処理×拡張心×普及理論でメッセージ波形と媒体を最適化。
- 美術品販売: ブルデュー/制度的芸術理論で正統性の足場を可視化、拡散は普及理論とシグナリングで。
