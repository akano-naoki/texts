# 有機体の維持から生まれる「欲望レイヤー」モデル
---

## 0. 目的

この文書は、「欲望」を **維持（環境変化を織り込んだ動的維持）** を実現するための **制御信号（ドライブ／方針）** として階層化したモデルを、議論参加者以外にも共有できる形に整理したものです。

- **低位（低級）**：短期・直接・反射に近い／媒介が少ない／脳の内側
- **高位（高級）**：長期・間接・モデル更新や規範／媒介が多い／脳の外側
- 階層は一方向の「上昇」だけでなく、**上位が下位を制御するフィードバック** も持つ（例：禁欲、断食、使命感）。

> 重要：階層は「順位」ではなく **条件（発火しやすさ・依存関係）**。
> 低位が不安定だと高位は発火しにくい一方、危機では高位が低位を支えることもあります。

### 立場表明（このモデルの前提）

このモデルは「社会（的活動）をこういう前提で見る」という **作業仮説**であり、数学の定理のような証明可能性を主張しない。ただし、恣意的な断定に閉じないために、次の立場を明示する。

- **着想源**：生体が外乱込みで安定を保つという **動的維持**（ホメオスタシス／アロスタシス）の考え方。
- **基本仮説**：欲望は、その動的維持を成立させるための **制御信号（誤差信号に対する方針）** として階層化できる。
- **連続化の狙い**：個体の維持（呼吸・安全など）と、共同体の維持（規範・制度・文化など）を、スケール違いの「制御」として連続的に記述する。
- **観測上の立場**：社会組織（企業・国家・共同体など）を、生物そのものではないが「境界・代謝・記憶・免疫」を持つ **擬似的な生命（自己維持する制御系）** として扱う。これは責任主体を免罪するためではなく、現象を制御構造として読み解くための比喩である。
- **構造的根拠**：レイヤーの順序は、脳の **内側から外側への構造**（脳幹→辺縁系→皮質）と対応させている。これは「科学」ではなく「科学っぽさ」の担保であり、OSI参照モデルのような階層的依存関係を説明するための補助線である。

### 評価規約（"それなりの正しさ"を保つために）

本モデルは、次の規約に基づいて改善される。

1. **反例が出たら更新する**（教義化しない）。
2. **説明／予測／設計に役立つか** で評価する（便利さを検証可能な価値として扱う）。
3. 同じ対象を扱う他モデルと比べ、**説明が短く・混線が少なく・共有しやすい** かを重視する。

## 1. レベリング方針：欲望は「起源（誤差信号）」でレベルを決める

同じ行動でも、**何が鳴っているか（起源＝誤差信号）**で所属レイヤーが変わり得ます。
このモデルでは、**欲望レイヤー（L）を「行動」ではなく「起源」で決める**ことを明示します。

- **起源（レベル判定の対象）**：いま何を維持しようとしているか／どんな誤差が鳴っているか
- **戦略（行動）**：その誤差を埋めるために選ばれた具体行動（道具・制度・計画も含む）
- **自覚**：本人がその誤差を意識できるか（起源と自覚は一致しないことがある）

## 2. 補助タグ

### 2.1 スコープ：I / We（Weなら広さ）

- **I**：個体の維持
- **We-近**：場・対面の関係（家族、恋人、チーム、界隈）
- **We-中**：共同体・組織（会社、学校、地域など）
- **We-広**：制度圏・文化圏（国家、宗教圏、文明、世代）
- **I→We(橋)**：個体内の状態を共有可能な形へ変換して橋渡し

### 2.2 モード：Now / Better

- **Now**：今の破綻回避・可動域復帰
- **Better**：将来適応・不確実性低減・長期維持
- **Now→Better**：短期対応が長期改善にもつながる

### 2.3 頻度（タイムスケール）

各レイヤーに、**起源信号が強く立ち上がる代表スケール**を添えます（目安）。

### 2.4 強度と閾値

同じレイヤーの欲望でも、**誤差信号の強さには個人差がある**。

- **強度**：同じ刺激に対して、どのくらい強く鳴るか
- **閾値**：どのくらいの刺激で鳴り始めるか

強度・閾値の違いは、神経系の特性による個人差であり、それ自体は善悪ではない。「平均」との差が弊害になる（例：ADHD的特性＝一部の欲望が弱く出る、HSP的特性＝一部の欲望が強く出る）。

## 3. Tier構造：脳の内側から外側へ

レイヤーは5つのTier（層）にグループ化される。これは脳の内側から外側への構造と対応する。

| Tier | 名称 | 脳構造 | レイヤー | 時間軸 |
|------|------|--------|----------|--------|
| **T0** | 基盤層 | 細胞膜・代謝 | L0 | — |
| **T1** | 反射層 | 延髄・橋・脊髄 | L1, L2, L3 | 秒〜日 |
| **T2** | 生存層 | 扁桃体・視床下部・PAG | L4, L5, L6 | 瞬時〜週 |
| **T3** | 関係層 | 社会脳・ミラー系・オキシトシン系 | L7, L8, L9 | 分〜年 |
| **T4** | 予測層 | 前頭前皮質・海馬・言語野 | L10, L11, L12 | 週〜年 |
| **T5** | 物語層 | DMN・顕著性ネットワーク | L13, L14 | 月〜世代 |

### Tier間の依存関係

```
T1が壊れると → T2以上が機能しない（呼吸が止まれば死ぬ）
T2が不安定だと → T3以上が発火しにくい（飢餓・恐怖下で所属を考えられない）
T3が不安定だと → T4以上が発火しにくい（孤立していると蓄積・探索に向かない）
T4が不安定だと → T5が発火しにくい（備えがないと規範を考える余裕がない）
```

**ただし、逆方向のフィードバックもある。**

```
T5（規範・使命）がT2（生存）を上書きする例 → 殉教、ハンガーストライキ
T3（所属）がT4（蓄積）を駆動する例 → 仲間のために学ぶ
```

## 4. 欲望スタック（起源ベース）：L0〜L14

以下では、各レベルを「起源（誤差）」で定義し、具体行動は「戦略例」として添えます。

---

### T0：基盤層

#### L0：前提層（欲望なし）— 物理法則・化学反応（散逸／平衡化）

- **タグ**：`（非生命）` / `—` / `頻度: —`
- **対応構造**：骨格、結合組織、細胞外マトリクス
- **起源（誤差）**：なし（目標域がない）
- **要点**：放置すると、摩擦・拡散・反応により勾配がならされ、構造がほどける（散逸）。
- **直感例**：温度差が消える／混ざる／錆びる／乾く／崩れる。

---

### T1：反射層（脳幹・脊髄）

#### L1：呼吸・飲水欲（最短ループ入力）

- **タグ**：`I` / `Now` / `頻度: 秒〜分（呼吸）、時間（飲水）`
- **対応構造**：延髄（呼吸中枢）、視床下部（渇中枢）
- **起源（誤差）**：酸素・水・電解質の欠乏（神経発火の三要素）
- **自覚される形**：息苦しさ、渇き
- **戦略例（行動）**：呼吸を整える／水を飲む／換気／場所移動／塩分補給
- **補足**：電解質の乱れは直接自覚しにくく、L8（調律）の不調として現れることが多い（だるさ、脱力、「なんか変」）

#### L2：排泄欲（出力・浄化）

- **タグ**：`I` / `Now` / `頻度: 時間`
- **対応構造**：脊髄反射、仙髄、橋（排尿中枢）
- **起源（誤差）**：内部の過剰・不要物・内圧が閾値を超える
- **自覚される形**：尿意、便意、吐き気
- **戦略例（行動）**：排尿・排便／嘔吐／発汗／衛生行動

#### L3：回復・保守欲（睡眠・休息）

- **タグ**：`I` / `Now→Better` / `頻度: 日（＋分〜時間の小休止）`
- **対応構造**：視床下部（視交叉上核）、松果体、脳幹網様体
- **起源（誤差）**：修復・統合・再起動の不足（疲労・回復遅延）
- **自覚される形**：眠気、疲労感
- **戦略例（行動）**：睡眠／休息／休暇／病欠／回復ルーティン
- **補足**：「休息」は"何の誤差を埋めるか"で起源が変わる：修復目的→L3、調律目的→L8

## 5. Tier境界：レベルが成立する前提

各Tierの境界は、新しい維持対象と誤差信号が成立する条件を示す。

| 境界 | 位置 | 何が成立するか |
|------|------|----------------|
| **T0↔T1** | L0とL1の間 | 境界（膜）＋代謝＋負帰還制御によって「自己維持」が立ち上がる。L1以降の「欲望（制御信号）」が意味を持つ。 |
| **T1↔T2** | L3とL4の間 | 全身スケールの統合維持（多細胞化）。脅威検出、補給、繁殖が独立した誤差信号として成立。 |
| **T2↔T3** | L6とL7の間 | 社会性の登場。個体間の長期関係が維持対象になる（愛着・共感・協力）。群れからの排除＝死という誤差信号が成立。 |
| **T3↔T4** | L9とL10の間 | 戦略性の登場。反射中心→学習→予測→計画（モデルを介した制御）。「備えが足りない」「地図が足りない」が独立した誤差として成立。 |
| **T4↔T5** | L12とL13の間 | 文化（象徴・儀礼・物語）の成立により、外化（L12）が共同体で回り始める。さらに分業と外部記憶（記録・貨幣・法）が維持装置として成立し、規範・制度（L13）が長期維持の主戦場になる。 |
